Getting Started:

1. Clone the Repo:

- git clone https://github.com/ibrahem-tech/Tic-Tac-Toe.git

2. Install the dependencies:

- cd tic-tac-toe-online
- npm install

3. Run the server:

- npm run server

4. Run the client:

- npm run client

5. Open your browser and navigate to http://localhost:3000/

6. Start Playing
