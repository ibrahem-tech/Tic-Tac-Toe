Getting Started:

Clone the Repo:
,,,
- git clone https://github.com/ibrahem-tech/Tic-Tac-Toe.git
,,,
Install the dependencies:
,,,
- cd tic-tac-toe-online
- npm install
,,,
Run the server:
,,,
- npm run server
,,,
Run the client:
,,,
- npm run client
,,,
Open your browser and navigate to http://localhost:3000/

- Start Playing
